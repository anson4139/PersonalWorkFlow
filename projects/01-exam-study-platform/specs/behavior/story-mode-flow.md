## 決策紀錄（全部確認）

| 項目 | 決定 |
|------|------|
| 科目選擇入口 | 首頁各 session 下，有 4 個模式按鈕（1題 / 10題 / 對戰 / **劇情**） |
| 地圖關卡數 | 3 關（普通 → 強化 → BOSS） |
| 怪獸抽取方式 | **進試煉地圖時一次抽定**，3 隻怪顯示在地圖，玩家可先看到是誰 |
| 怪獸池 | 普通怪池 1–10、強化怪池 11–20、BOSS 池 21–25，各隨機抽 1 |
| 題數比例 | 普通 2 : 強化 3 : BOSS 5（session 50 題 → 10/15/25 題） |
| 章節開場 + 關卡介紹 | **保留**（C + D 畫面），可稍後依試用結果調整 |
| 結果頁順序 | BattlePage 結果頁 → 點「← 返回」→ 劇情通關/落敗畫面 |
| 影像2 bug | 需修除，不應出現 BattlePage 的選科目 UI |
| 通關條件 | 3 關全通觸發大結局演出 |

<summary>## 2. 劇情模式完整流程樹（最終版）</summary>

```
首頁
  └─ 選好科目 + session，點「劇情模式」
       │
       ▼
╔══════════════════════════════════════╗
║  A. 故事序章（僅第一次進入時顯示）   ║
╚══════════════════════════════════════╝
  「深淵試煉地圖已在眼前展開。
   3 道試煉等待著你。
   普通怪、強化怪、BOSS——
   用你的知識，逐一擊敗它們！」
  → 點「開始試煉」（localStorage 記錄，不再重播）
       │
       ▼
╔══════════════════════════════════════╗
║  B. 試煉地圖（進圖時一次抽定 3 隻）  ║
╚══════════════════════════════════════╝
  進入地圖時，程式自動：
    從普通怪池 (1–10) 隨機抽 1 隻 → 顯示在格子 1
    從強化怪池(11–20) 隨機抽 1 隻 → 顯示在格子 2
    從 BOSS 池   (21–25) 隨機抽 1 隻 → 顯示在格子 3
  玩家可在地圖看到這 3 隻怪獸是誰，再決定點哪關

  格子狀態：
    ✓ 通關（綠框）
    ✗ 待重打（紅框）
    ⚔ 可挑戰（閃爍綠框）
    ? 未解鎖（暗色）
  │
  └─ 點已解鎖的關卡格子
       │
       ▼
╔══════════════════════════════════════╗
║  C. 章節開場（第一次進該關才顯示）   ║
╚══════════════════════════════════════╝
  第 1 關（普通怪）：「試煉開始。用你的知識熱身吧！」
  第 2 關（強化怪）：「你通過了初試。前方的敵人更強大了。」
  第 3 關（BOSS）  ：「最終決戰。深淵的主宰正在等待你！」
  → 點「繼續」
       │
       ▼
╔══════════════════════════════════════╗
║  D. 關卡介紹（怪獸大圖 + 台詞）      ║
╚══════════════════════════════════════╝
  顯示本關抽到的怪獸大圖 + 光效
  怪獸台詞（可用通用格式）：
    「你敢來挑戰我？用你的知識打敗我！」
  ├─ 點「⚔ 開始戰鬥」→ 進 E
  └─ 點「← 返回地圖」→ 回 B
       │
       ▼
╔══════════════════════════════════════╗
║  E. 戰鬥中（BattlePage）             ║
╚══════════════════════════════════════╝
  題庫：首頁選定的 subjectKey + session
  出題數（2:3:5 比例，session 50 題為例）：
    第 1 關 普通怪 → 10 題
    第 2 關 強化怪 → 15 題
    第 3 關 BOSS   → 25 題
  BattlePage 不顯示選科目畫面（由 StoryPage 傳入 initialKey + initialSession + questionCount）
  │
  ├─ [✕ 中途放棄] → 回 B（不記錄）
  └─ [所有本關題目答完]
       │
       ▼
╔══════════════════════════════════════╗
║  F. 戰鬥結果頁（BattlePage result）  ║
╚══════════════════════════════════════╝
  分數 / 連擊 / 擊敗怪獸數
  → 點「← 返回」
       │
       ▼
╔══════════════════════════════════════╗
║  G. 通關 / 落敗劇情（StoryPage）     ║
╚══════════════════════════════════════╝
  ── 通關（分數 >= 60）──
  「你用知識的力量，擊敗了 {怪獸名稱}！」
  「下一關已解鎖。」
  → 點「繼續」→ 回 B（格子更新 ✓，下一格解鎖）

  ── 落敗（分數 < 60）──
  「{怪獸名稱}：你的知識還不夠，再來一次！」
  「但你已踏出了一步——下一關仍然為你開啟。」
  → 點「繼續」→ 回 B（格子更新 ✗，下一格解鎖）

  ── 3 關全通（BOSS 擊倒後）──
  大結局演出：
  「所有試煉，已被你的知識征服。」
  「深淵的主宰，是你。」
  → 點「回首頁」→ 首頁
```

<summary>## 3. 怪獸池分類與題數計算</summary>

### 怪獸池

| 池子 | 怪獸編號 | 包含怪獸 |
|------|---------|---------|
| 普通怪池 | 1–10 | 火焰龍、石巨人、暗黑騎士、雷鷹、史萊姆球、毛球熊、惡作劇狐、暗影惡靈、骷髏王、深淵蛇妖 |
| 強化怪池 | 11–20 | 熔岩龍王、鐵甲巨人、死神騎士、風暴鷹神、毒液之王、怒熊霸主、九尾狐妖、幽冥虛空主、不死帝王、深淵巨龍 |
| BOSS 池 | 21–25 | 末日火神、大地終結者、混沌魔神、雷霆至尊、深淵神明 |

### 題數分配

| 關卡 | 比例 | 50 題 session | 25 題 session |
|------|------|--------------|--------------|
| 第 1 關 普通怪 | 2/10 | 10 題 | 5 題 |
| 第 2 關 強化怪 | 3/10 | 15 題 | 8 題（四捨五入）|
| 第 3 關 BOSS | 5/10 | 25 題 | 13 題（四捨五入）|

捨入規則：`Math.round(total * ratio)`，最後一關取餘數補齊。

- 各 session 項目下加「劇情模式」按鈕

- 接收 props：`subjectKey: SubjectKey`、`sessionName: string`、`onBack: () => void`
- 進圖時從三個怪獸池各隨機抽 1 隻，存入 state

- `storyPhase`：`prologue` | `map` | `chapter-intro` | `stage-intro` | `battle` | `stage-clear`
- 依關卡編號計算 questionCount 傳給 BattlePage

- 新增 prop：`questionCount?: number`（限制本關出題數）
- 當 `initialKey` + `initialSession` + `questionCount` 全部有值時，跳過選科目畫面直接開戰
